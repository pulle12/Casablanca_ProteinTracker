# User Story 3

## Story
As a user, I want to add a meal with its protein amount so that my daily intake updates.

## Acceptance Criteria
- The user can add a meal name.
- The user can add protein amount for that meal (in grams).
- After adding, daily consumed protein updates immediately.
- Invalid values (empty name, negative numbers) are blocked.

## Definition of Done
- New meal appears in the daily meal list.
- Daily total is recalculated correctly.
- User gets simple feedback after adding a meal.

## Priority
Must Have
