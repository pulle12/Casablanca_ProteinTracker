# User Story 5

## Story
As a user, I want my meals to be stored in a JSON file so that my meal list stays available.

## Acceptance Criteria
- New meals are written to the JSON file.
- Existing meals stay in the file after app restart.
- Meal data includes at least name and protein value.
- If saving fails, the user sees a clear error message.

## Definition of Done
- JSON file structure is documented and consistent.
- Save and load are tested with at least 5 meals.
- No duplicate meal entries are created by mistake.

## Priority
Must Have
