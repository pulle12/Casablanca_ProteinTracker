# User Story 1

## Story
As a user, I want to enter my weight and height so that the app can calculate my daily protein target.

## Acceptance Criteria
- The user can enter weight in kg.
- The user can enter height in cm.
- The app validates both inputs (numbers only, greater than 0).
- After valid input, the app calculates and shows a daily protein target.

## Definition of Done
- Input fields are visible and usable on desktop and mobile.
- Validation messages are understandable for beginners.
- Calculation is tested with at least 3 example values.

## Priority
Must Have
