# User Story 4

## Story
As a user, I want to choose meals from a saved meal list so that I do not need to re-enter common meals every time.

## Acceptance Criteria
- The app shows a list of saved meals from the JSON file.
- The user can select one meal from the list.
- After selection, the meal is added to today's intake.
- The daily total updates automatically.

## Definition of Done
- Meal list loads without app crash.
- Selected meal adds correct protein value.
- If JSON is empty, the user sees a friendly message.
