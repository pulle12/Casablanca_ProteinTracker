# User Story 2

## Story
As a user, I want to see my daily protein target clearly so that I know how much protein I should eat.

## Acceptance Criteria
- The daily target is shown directly after calculation.
- The target value is displayed in grams per day.
- The target is easy to find on the screen (clear label and value).
- If no user data is entered yet, the app shows a helpful hint.

## Definition of Done
- Target text is readable and not hidden in the UI.
- The shown value matches the calculation logic.
- A short hint explains what the target means.

## Priority
Must Have
