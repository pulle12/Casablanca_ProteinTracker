# User Story 6

## Story
As a user, I want to see a chart of consumed protein vs target so that I can track progress quickly.

## Acceptance Criteria
- The chart shows consumed protein and target protein.
- The chart updates after adding a meal.
- Values in the chart match the displayed totals.
- The chart is readable on desktop and mobile.

## Definition of Done
- Chart appears without manual page reload.
- Axis labels or legends are understandable.
- Tested with 0 g, half target, and over target values.
