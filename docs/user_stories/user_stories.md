# User Stories

1. As a user, I want to enter my weight and height so that the app can calculate my daily protein target.
2. As a user, I want to see my daily protein target clearly so that I know how much protein I should eat.
3. As a user, I want to add a meal with its protein amount so that my daily intake updates.
4. As a user, I want to choose meals from a saved meal list so that I do not need to re-enter common meals every time.
5. As a user, I want my meals to be stored in a JSON file so that my meal list stays available.
6. As a user, I want to see a chart of consumed protein vs target so that I can track progress quickly.
