# Prompt for PRD Creation

You are a Product Manager. Turn the following product idea into a short and clean PRD for a student project.

The PRD should include:
- Product Name
- Problem
- Solution
- Target Users
- Core Features
- Non-Goals
- Simple Tech Suggestion

Keep the wording simple and beginner-friendly.

Product Idea:
A protein tracker that tracks daily protein intake and gives the user an opportunity to enter body weight and height, then calculates the daily protein goal. In the background, meals are saved in a JSON file. The user can choose a meal from the file and add those meals to the daily intake. It should show the user how much protein is needed in a chart based on weight and height.

Also suggest a meaningful group name for the whole project.
